# Optimierung von JavaScript-Rendering

Das Rendern von JavaScript kann die Performance einer Webseite erheblich beeinflussen. In diesem Artikel werden wir uns verschiedene Strategien zur Optimierung ansehen.

## 1. Vermeidung von Reflows

Ein Reflow tritt auf, wenn der Browser das Layout einer Seite neu berechnet. Häufige Ursachen sind:

- Direkte Manipulation von Stileigenschaften (`element.style.height = "100px"`)
- Zugriff auf `offsetWidth`, `clientHeight` usw., gefolgt von Änderungen am DOM

Lösung: **CSS-Änderungen bündeln und `requestAnimationFrame` nutzen.**
